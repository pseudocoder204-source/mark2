# Contributing

## Code

**This project is not accepting code contributions right now.** Pull requests that change
source files will be closed unread. This is not about the quality of your work — it is
that Pulser is authored entirely by one person, and keeping it that way preserves the
option to relicense or commercialise it later without tracking down past contributors.

Bug reports and feature ideas are welcome and useful. Open an issue.

If this policy changes, a Contributor License Agreement will be added here first, and
pull requests will require signing it. Until that file exists, assume code PRs are not
being merged.

## Scan data

Contributing **scan data** to improve the report model is welcome and is a separate
process with its own rights grant. See
[`CONTRIBUTING_SCAN_DATA.md`](CONTRIBUTING_SCAN_DATA.md).

## Security issues

Do not open a public issue for a vulnerability in Pulser itself. Open a GitHub security
advisory on the repository instead.
